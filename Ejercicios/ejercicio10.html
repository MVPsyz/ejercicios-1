HTML, cración de páginas web
y términos relacionados

HTML:
HTML es el código que se utiliza para estructurar y desplegar una página web y sus contenidos.

CSS:
CSS es un lenguaje para la composición y estructuración de páginas web (HTML o XML).

Cite:
Es una etiqueta de texto HTML.

Etiqueta:
