Lenguajes de Programación y de Marca


Lenguajes de Marca:


Es la forma de codificar un documento o página web, es una manera de definir la estructura del texto o su presentación incorporadon etiquetas o marcas que contienen información adicional.



Lenguajes de Programación:


Es una herramienta informática que permite desarrollar programas para PCs.



Ejemplos:


Ejemplo de lenguaje de Marca;HTML: Lenguaje de Marcado de Hipertexto, es el código que se utiliza para estructurar y desplegar una página web y sus contenidos.
Lenguaje de Programación; Bourne-Again Shell es un intérprete de comandos y lenguaje de programación integrado que corre bajo el macroprocesador Shell de Unix.


W3Schools; página web donde puedes encontrar tanto leguajes de marca, como de programación para aprender.
