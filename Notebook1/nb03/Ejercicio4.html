Etiqueta (Tag):
Es un componente fundamental de HTML que se usa para definir la estructura y el contenido de una página web. Las etiquetas van entre < > y suelen tener una apertura y un cierre, como "p", que indica un párrafo.
Atributo:
Es una propiedad adicional que proporciona información extra sobre una etiqueta. Se escribe dentro de la etiqueta de apertura y modifica su comportamiento. Por ejemplo, en "a href="https://example.com">, el atributo href especifica el enlace al que se dirige.
DOCTYPE:
Es una declaración al inicio de un documento HTML que indica al navegador qué versión de HTML se está utilizando. En HTML5, se escribe simplemente como y asegura que el navegador interprete correctamente el código.
